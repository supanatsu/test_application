# 💖 For My Dear Web App

แอปพลิเคชันเว็บส่วนตัวสุดโรแมนติกที่สร้างขึ้นมาเพื่อ "เดียร์" โดยเฉพาะ เป็นพื้นที่เล็กๆ สำหรับบอกรัก เช็คความรู้สึก ติดตามภารกิจประจำวัน และแชร์ช่วงเวลาดีๆ ด้วยกัน

## ✨ ฟีเจอร์หลัก (Features)

* 💌 **Love Messages & Stats (บอกรักและสถิติ):** * สุ่มข้อความบอกรักสุดน่ารัก
    * นับจำนวนวันที่คบกัน (Days Together)
    * นับจำนวนครั้งที่กดส่งข้อความบอกรัก
* ✅ **Daily Checklist (ภารกิจเติมความหวาน):** * ติดตามกิจวัตรประจำวัน เช่น ดื่มน้ำ กินข้าว พักผ่อน หรือบอกรัก
    * มีแถบแสดงความคืบหน้า (Progress Bar) และบันทึกข้อมูลแบบวันต่อวัน
* 📊 **Mood Tracker (บันทึกอารมณ์):** * ให้เดียร์บันทึกความรู้สึกในแต่ละวัน (😄, 🙂, 😮‍💨, 😢)
    * แสดงผลอารมณ์ย้อนหลัง 7 วันในรูปแบบกราฟโดนัท (Doughnut Chart)
* 📍 **Location Sharing (ดูตำแหน่ง):** * ดูพิกัดปัจจุบันของเดียร์แบบ Real-time พร้อมปุ่มเปิดใน Google Maps (ใช้ Geolocation API)
* 📺 **Screen Sharing (แชร์หน้าจอ):** * แชร์หน้าจอหรือแท็บเบราว์เซอร์เพื่อดูหนังหรือทำกิจกรรมร่วมกัน (ใช้ WebRTC)
* 📱 **PWA Ready (รองรับการติดตั้งเป็นแอป):** * สามารถกดติดตั้งลงบนหน้าจอโฮมของมือถือได้ (Install Prompt)
    * มีระบบเช็คสถานะ ออนไลน์/ออฟไลน์

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** HTML5, Vanilla JavaScript
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (via CDN)
* **Data Visualization:** [Chart.js](https://www.chartjs.org/) (via CDN)
* **Storage:** Local Storage (สำหรับเก็บข้อมูล Checklist และอารมณ์)
* **APIs:** Geolocation API, MediaDevices API (Screen Sharing)

## 🚀 วิธีการติดตั้งและใช้งาน (Getting Started)

1.  **โคลนโปรเจกต์ (Clone the repository):**
    นำไฟล์ทั้งหมดไปไว้ในโฟลเดอร์โปรเจกต์ของคุณ
2.  **ไฟล์ที่จำเป็นต้องมี:**
    เพื่อให้ระบบ PWA ทำงานได้สมบูรณ์ กรุณาสร้างไฟล์เหล่านี้เพิ่ม (อ้างอิงจาก `<head>` ใน `index.html`):
    * `manifest.json`
    * `service-worker.js`
    * `icon-192.png` (ไอคอนแอป)
3.  **การรันโปรเจกต์:**
    * สามารถเปิดไฟล์ `index.html` บนเบราว์เซอร์ได้โดยตรง
    * **ข้อควรระวัง:** ฟีเจอร์ **Geolocation** และ **Screen Sharing** จำเป็นต้องรันบนเซิร์ฟเวอร์ที่มี `https://` หรือรันผ่าน `localhost` เท่านั้น (แนะนำให้ใช้ Live Server extension ใน VS Code)

## 🎨 การปรับแต่ง (Customization)

คุณสามารถแก้ไขโค้ดใน `index.html` เพื่อปรับแต่งข้อมูลส่วนตัวได้:

* **เพิ่มคำบอกรัก:** ไปที่บรรทัด `const messages = [...]` ในส่วนของ JavaScript และเพิ่มข้อความของคุณลงใน Array
* **ตั้งคหาวันเริ่มคบกัน (Start Date):** ค้นหาฟังก์ชัน `updateStats()` และเปลี่ยนวันที่ใน `const startDate = new Date('2024-01-01');` เป็นวันครบรอบของคุณ

## 🔒 ความเป็นส่วนตัว (Privacy Notice)
ข้อมูลทั้งหมด (Checklist, ความรู้สึก, จำนวนครั้งที่บอกรัก) จะถูกบันทึกไว้ใน **Local Storage** ของเบราว์เซอร์ในเครื่องผู้ใช้เท่านั้น ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก เพื่อความเป็นส่วนตัวสูงสุดของคู่รัก
